Features

Everything you love,
now extra cute

Built on Ghostty's blazing-fast foundation with a cute aesthetic that makes your terminal feel like home.

⚡

GPU Accelerated

Blazing fast rendering powered by Metal on macOS. Buttery smooth scrolling and instant response, no matter how cute it looks.

🎨

Native macOS UI

Built with platform-native technologies. Proper window management, native tabs, split panes, and system integration that feels right.

🌈

Cute by Default

Pre-configured with a pastel color palette and cute aesthetic touches. Beautiful out of the box, no config files needed.

🔀

Splits & Tabs

Powerful window management with split panes and tabs. Organize your workspace however you like with keyboard shortcuts.

🔌

Customizable

Fully configurable through a simple config file. Fonts, colors, keybindings, and more. Make it yours while keeping it cute.

🐦

Open Source

Free and open source, built on the amazing Ghostty project. Contribute, customize, and share the cuteness with everyone.

Install

Up and running in seconds

Getting Cute Ghostty on your Mac is as easy as downloading, unzipping, and falling in love.

1

Download

Grab the latest release from GitHub. It's a single zip file.

CuteGhostty.zip
2

Unzip & Move

Unzip the file and drag Cute Ghostty.app into your Applications folder.

/Applications/
3

Launch

Right-click and select "Open" on first launch. After that, just double-click like normal.

open -a "Cute Ghostty"

Requirements

 macOS 13.0+ (Ventura)
⚙ Apple Silicon (arm64)
