# Cute Ghostty

A cute customized build of [Ghostty](https://ghostty.org), the fast and feature-rich terminal emulator.

## Download

Download the latest release from the [Releases](https://github.com/amywork777/CuteGhostty/releases) page.

## Install

1. Download `CuteGhostty.zip` from Releases
2. Unzip the file
3. Drag **Cute Ghostty.app** to your Applications folder
4. On first launch, right-click the app and select "Open" to bypass Gatekeeper

## Requirements

- macOS 13.0 (Ventura) or later
- Apple Silicon (arm64)

## About

Cute Ghostty is a customized build of Ghostty with a cute aesthetic. Ghostty is a fast, feature-rich, and cross-platform terminal emulator that uses platform-native UI and GPU acceleration.
